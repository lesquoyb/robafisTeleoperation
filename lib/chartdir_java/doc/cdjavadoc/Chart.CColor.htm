ChartDirector 6.0 (Java Edition)

Chart.CColor

Usage

public static int CColor(java.awt.Color color)

Description

Converts a java.awt.Color object to a ChartDirector color.

ChartDirector represents ARGB colors using a web and HTML compatible format. For example, red is represented as FF0000, the same as used in HTML.

On the other hand, in Java, colors are sometimes represented as java.awt.Color objects. This method converts a java.awt.Color object to ChartDirector's color format.

Arguments

ArgumentDefaultDescription
color(Mandatory)A java.awt.Color object.

Return Value

A ChartDirector color.
