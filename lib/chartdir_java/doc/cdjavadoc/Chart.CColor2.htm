ChartDirector 6.0 (Java Edition)

Chart.CColor2

Usage

public static int[] CColor2(java.awt.Color[] colors)

Description

Converts an array of java.awt.Color objects to an array of ChartDirector colors.

This function is similar to Chart.CColor, but operate on an array of java.awt.Color objects instead of a single java.awt.Color object.

Arguments

ArgumentDefaultDescription
colors(Mandatory)An array of java.awt.Color objects.

Return Value

An array of ChartDirector colors.
