ChartDirector 6.0 (Java Edition)

Chart.CTime2

Usage

public static double[] CTime2(java.util.Date[] times)

Description

Converts an array of java.util.Date objects to ChartDirector's date/time format.

This function is similar to Chart.CTime, but operate on an array of java.util.Date objects instead of a single java.util.Date object.

Arguments

ArgumentDefaultDescription
times(Mandatory)An array of java.util.Date objects.

Return Value

An array of numbers representing dates/times in ChartDirector's date/time format.
