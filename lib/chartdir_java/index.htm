Check ChartDirector JSP Installation
If you have installed ChartDirector on a JSP web server, you can click on the icon to check if the ChartDirector installation is working properly.
ChartDirector Documentation
Click on the icon for the ChartDirector HTML documentation. It contains numerous tutorials with source code listing, complete ChartDirector API reference, and instructions on how to install ChartDirector in your own web application.

The same documentation is also available in CHM format from the ChartDirector download page. The CHM format has better search and index facilities.
ChartDirector Sample Code
For JSP installation, you can browser and run all JSP sample code by clicking on the icon. For standalone usage, please refer to the documentation on how to compile and run the sample code.
